What is the difference in meaning between the following pairs (a/b and c/d) of groups of C statements: 

a. 
```c
if (x == 0) {
    printf("zero\n");
}
```

b. 
```c
if (x == 0)
    printf("zero\n");
```
c. 
```c
if (x == 0) {
    printf("zero\n");
    printf("after\n");
}
```
d. 
```c
if (x == 0)
    printf("zero\n");
    printf("after\n");
```
